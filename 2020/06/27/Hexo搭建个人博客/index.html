Hexo搭建个人博客

发表于 2020-06-27 更新于 2020-07-07 阅读次数： 阅读次数： Valine：

（默认安装了Git，安装Git的操作在我的博客文章Git的安装使用中查看）

1. 下载安装Node.js

①进入官网下载：https://nodejs.org/en/


②双击安装包安装



③选择安装路径




④完成安装

⑤Windows+R输入cmd打开命令行窗口

​ 执行”node -v” “npm -v”指令查看当前版本

2. 安装Hexo

①利用淘宝镜像源安装cnpm

​ 执行指令：npm install -g cnpm –registry=https://registry.npm.taobao.org

​ 执行指令：cnpm -v查看cnpm当前版本

②利用cnpm安装Hexo

​ 执行指令：cnpm install -g hexo-cli

​ 执行指令：hexo -v查看hexo当前版本

③在此路径下新建一个blog文件夹，对Hexo进行初始化

​ 执行指令：hexo init

​ 由于我在此出现了报错，所以改用在blog文件夹窗口处右键启动Git Bash Here进行安装

​ 初始化完成后blog文件中的内容

④新建一篇文章，名称为”我的第一篇文章“

⑤启动Hexo本地博客

​ 执行指令：hexo s

​ 浏览器中输入：http://localhost:4000/访问本地博客，这样就可以看到博客的初步效果

⑥用编辑器打开blog文件夹中的source_post\我的第一篇文章.md进行编写（后面会介绍Typora编辑器）

⑦生成博客

执行指令：hexo g


3. 将博客部署到GitHub上

①在GitHub上创建一个新仓库

​ 仓库名必须是：用户名.github.io（这里由于我之前已经创建过一次，所以显示是红色报错）

②在blog路径下安装一个git部署的插件

​ 执行指令：cnpm install –save hexo-deployer-git

③修改blog文件夹中的_config.yml文件

​ 在文件最后的位置做如下修改并保存

​ 注意：“：”后有一个空格

④将博客部署到远端

​ 执行指令：hexo d

⑤在浏览器中输入博客仓库的地址访问博客

4.更换Hexo博客主题

①在GitHub上找一些大佬制作的主题（这里推荐主题yilia：https://github.com/litten/hexo-theme-yilia）

​ 执行指令：git clone https://github.com/litten/hexo-theme-yilia.git themes/yilia

​ 这时blog/theme文件夹中就会出现yilia文件夹

②将yilia主题配置到博客

​ 修改配置文件_config.yml，将theme landscape→theme: yilia，保存

③重新生成博客

主题推荐：https://github.com/jinyaoMa/hexo-theme-mustom

更多主题可去官网查看：https://hexo.io/themes/
